// CV Interactivo Web // 

Este proyecto consiste en el desarrollo de un currículum vitae digital interactivo, diseñado como una página web moderna, en donde uno pueda presentarse a los posibles clientes de forma llamativa y atractiva.

El objetivo de este sitio es presentar mi información profesional de forma moderna, para así mejorar la experiencia del usuario al destacar habilidades en desarrollo wen Front-end.

El sitio está construido utilizando HTML, CSS, JavaScript, jQuery y Bootstrap; en el cual se incorporó animaciones, efectos visuales y funcionalidades dinámicas como modo claro/oscuro, navegación suave y validación de formularios en tiempo real.

Entre sus principales características destacan:

Diseño responsivo adaptable a distintos dispositivos
Navegación mediante navbar interactivo
Secciones organizadas en formato de tarjetas
Animaciones al hacer scroll (AOS)
Modo oscuro/claro con cambio dinámico
Barras de habilidades con efectos interactivos
Formulario de contacto con validación en tiempo real
